# Employee Management System

## Project Type
Training / Academic Project

## Description
Employee Management System is a full-stack application designed to manage
employee records efficiently. The system supports adding, updating, viewing,
and deleting employee information.

This project was developed as part of Java Full Stack training and focuses on
understanding real-time application architecture and development flow.

## Features
- Employee CRUD Operations
- REST API based architecture
- Database integration
- Basic role-based access (conceptual)

## Tech Stack
- Java 8
- Spring Boot
- ReactJS
- MySQL
